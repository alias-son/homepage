모델 학습을 위해 만들어진 여러 py파일 리스트로 지정  

```
import os
import subprocess

# py파일을 리스트로
model_scripts = ['a.py','b.py','c.py','d.py']

def run_model(script_name):
    subprocess.run(['python', script_name])

for script in model_scripts:
    print(f"Running {script}...")
    run_model(script)
    print(f"Finished {script}\n")
```
